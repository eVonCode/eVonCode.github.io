✦ Featured Project

Gamified 3D Study Environment with AI Virtual Assistant

For Junior High School English Learners

🎮 Godot 3 🤖 AI-Powered 📚 EdTech 🏫 JHS Students

An ongoing development interactive 3D educational platform designed to make English learning engaging for Grades 7–10 students. Built in Godot 3 with an AI-powered virtual assistant, game-based activities, and an in-game PDF viewer — connected to a Python/Flask backend.

Godot 3 GDScript Python Flask SQLite Groq API Blender Figma

Key Features

AI Virtual Assistant powered by the Groq API
Word Search, Flashcards & Fill-in-the-Blanks mini-games
In-game PDF Module Viewer via Python & Poppler
Grade 7–10 English lesson browser
AI-powered module upload validation
Google OAuth login & session management
